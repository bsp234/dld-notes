Step 1: State Diagram (3-bit Gray Code Counter)

Shows all states as circles and transitions as arrows. For a 3-bit Gray code counter, 8 states cycle in Gray code order:

000 → 001 → 011 → 010 → 110 → 111 → 101 → 100 → 000

Key property of Gray code: only 1 bit changes between consecutive states

For D FF: D = Q⁺ always! The D input simply equals the desired next state.

Combine the next-state table (Step 2) with the FF transition table (Step 3) to find required J & K inputs:

X = don't care (can be 0 or 1 — used for K-map simplification!)

Step 5: Karnaugh Maps

Plot each J and K column from the excitation table on a 3-variable K-map (variables: Q₂, Q₁, Q₀). Don't-cares (X) can be grouped as 0 or 1 — use them to form larger groups!

K-Map Tips
  • Largest possible groups of 1s (or X treated as 1) = simplest expression
  • Groups must be powers of 2: 1, 2, 4, 8…
  • Wrap-around groups are valid
  • One K-map per FF input (J₂, K₂, J₁, K₁, J₀, K₀)

After simplification: derive minimal SOP/POS expression for each input

Step 6: Logic Expressions (3-bit Gray, JK)

From K-map minimization of the excitation table:

J₂ = Q₁·Q̄₀      K₂ = Q₁·Q̄₀
J₁ = Q̄₂·Q₀      K₁ = Q₂·Q₀
J₀ = Q̄₂·Q̄₁ + Q₂·Q₁    K₀ = Q̄₂·Q̄₁ + Q₂·Q₁

For D Flip-Flop design (Exercise):

D₂ = Q₂⁺   D₁ = Q₁⁺   D₀ = Q₀⁺

D inputs directly equal next-state values — read from next-state table then K-map simplify!

D FF Results (3-bit Gray, from K-map)

D₂ = Q̄₂Q₁Q̄₀ + Q₂Q̄₁ + Q₂Q₀
D₁ = Q̄₂Q₁Q̄₀ + Q̄₂Q̄₁Q₀ + Q₂Q₁
D₀ = Q̄₂Q̄₁ + Q₂Q₁ (= Q₂ ⊕ Q₁ inverted)

Step 7: Counter Implementation

From the Boolean expressions, draw the circuit using:

  • J-K or D flip-flops (one per state bit)
  • AND / OR / NOT gates for the input logic
  • All flip-flops driven by the same clock
Circuit Check

Verify by tracing the state sequence from 000. Each clock pulse should advance to the correct next Gray code state.

Key Takeaways — Synchronous Counter Design
✓
The 7 steps are always the same regardless of count sequence or flip-flop type
✓
State diagram → circles = states, arrows = clock transitions
✓
JK transition: 0→1 requires J=1,K=X; 1→0 requires J=X,K=1; no change: 0→0 J=0,K=X or 1→1 J=X,K=0
✓
D FF is simpler — D input always = desired next state Q⁺
✓
Don't-cares (X) in excitation table come from JK transitions — exploit them in K-maps for simplest expression
✓
All flip-flops clocked together = synchronous. No propagation delay issues
