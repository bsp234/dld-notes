① Combinational Circuits

Definition: Output depends only on current inputs — no feedback, no memory.

Output = f(Input) — no history!
Circuit Analysis vs. Design:
AnalysisDesign
Given circuit → find function (Boolean / Truth Table)Given function → build circuit
Topics Covered:
Decoders Encoders MUX DEMUX Comparators Adders Subtractors BCD Adder
Design Procedure:
  1. Determine number of inputs/outputs
  2. Derive the truth table
  3. Simplify Boolean expression (K-map)
  4. Implement the circuit
② Decoders

Function: Extract information from a binary code. n inputs → 2ⁿ outputs. Exactly ONE output is HIGH at a time.

2-to-4 Line Decoder
I₁I₀Y₃Y₂Y₁Y₀
000001
010010
100100
111000

Y₀=I₁'I₀'   Y₁=I₁'I₀   Y₂=I₁I₀'   Y₃=I₁I₀

  • Enable (E): When E=0, all outputs = 0. When E=1, normal operation.
  • Expansion: Two 2-to-4 decoders + 1 inverter = 3-to-8 decoder.
  • Active-Low: Outputs are inverted (NAND gates). Active output = 0.
n-to-2ⁿ decoder: implements all n-variable minterms
BCD-to-Excess-3 Code Converter

Add 3 (0011) to each BCD digit using combinational logic with don't cares for invalid BCD codes (1010–1111).

w=A+BC+BD  |  x=B'C+B'D+BC'D'  |  y=C'D'+CD  |  z=D'

Full Adder (FA)

FA = two HAs + one OR gate.

Ripple Carry Adder — Problem!

4-bit ripple carry = 8 gate levels. N-bit = 2N levels. Slow!

Carry Look-Ahead (CLA) — Solution!
Pᵢ=AᵢBᵢ (propagate)  |  Gᵢ=AᵢBᵢ (generate)
Cᵢ₊₁ = Gᵢ + PᵢCᵢ (all carries in parallel!)

All carries computed simultaneously → faster!

⑦ BCD Adder, Subtractor & Overflow
BCD Adder

BCD addition: valid result = 0–9. If result > 9 or carry=1, add +6 (0110) to correct.

Sum S₃S₂S₁S₀CyError?
0000–10010No
1010–11110Yes → +6
Any1Yes → +6
Err = S₃S₂ + S₃S₁ + Cy

Two binary adders: First adds X+Y, second adds +6 if needed.

Binary Subtractor
x − y = x + (−y) = x + y' + 1

Use 2's complement: invert y bits and set Cᵢₙ=1 to binary adder.

Adder/Subtractor (Mode M):
MOperation
0F = x + y (ADD)
1F = x − y (SUB)

M=1: y bits XOR'd with M=1 inverts them; Cᵢₙ=M=1 → 2's complement.

Overflow Detection
Overflow = C₄ ⊕ C₃ (for 2's complement)

For unsigned: overflow = C₄ (carry out of MSB).

⑧ Comparator

Compares two binary numbers A and B. Outputs: A>B, A<B, A=B.

ConditionExpression
A = BAll bits equal: Eᵢ=AᵢBᵢ+Aᵢ'Bᵢ'
A > BMSB of (A−B) = 0, result positive
A < BMSB of (A−B) = 1, result negative
Key Takeaways
✓
Combinational circuits: no feedback, output = f(inputs only).
✓
Decoder: n→2ⁿ, one-hot output. Encoder: 2ⁿ→n, one-hot input.
✓
MUX: data selector. DEMUX: data distributor. They form transmission pairs.
✓
CLA adder eliminates ripple carry delay: all carries computed in parallel.
✓
BCD sum >9 or Cy=1 → add 6 to correct. Subtraction = 2's complement addition.
