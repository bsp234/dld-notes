① What is an IC?

An Integrated Circuit (IC) is a silicon semiconductor crystal (chip) containing the electronic components for constructing digital gates. Various gates are connected internally to form the required circuit.

  • Logic Gates are the basic building blocks of digital logic and circuits.
  • ICs are categorized by complexity — measured by the number of logic gates in a single package.
  • Each circuit technology is called a digital logic family.
💡 Remember

The basic circuit in each technology is a NAND, NOR, or Inverter gate. Each logic family builds all complex circuits from this one basic gate.

③ Positive & Negative Logic

Two conventions exist for assigning binary values to voltage levels:

Positive Logic (Active High)

Logic 1 = HIGH voltage (H)
Logic 0 = LOW voltage (L)
Most common convention.

Negative Logic (Active Low)

Logic 1 = LOW voltage (L)
Logic 0 = HIGH voltage (H)
Used in some special cases.

⚡ Key Insight: AND gate dual

The same physical circuit with positive logic implements AND, but with negative logic it implements OR! The truth table rows just get re-mapped.

④ IC Parameters (Overview)

Four key electrical parameters determine the performance and suitability of an IC family:

1
Fan-Out — How many gate inputs can one gate output drive?
2
Power Dissipation — Energy consumed per gate (affects heat & battery life)
3
Propagation Delay — Time for output to respond to input change (affects speed)
4
Noise Margin — Tolerance to unwanted electrical noise (affects reliability)

⚖️ Trade-off: High speed (low delay) usually means higher power dissipation. Low power (CMOS) trades some speed.

⑤ Fan-Out (Loading)

Fan-Out = the maximum number of standard loads (gate inputs) that can be connected to a gate output without degrading normal operation.

Fan-Out = Min(IOH/IIH , IOL/IIL)

IOH = output current (High state)  |  IIH = input current needed (High)
IOL = output current (Low state)  |  IIL = input current needed (Low)

Ex: Standard TTL Fan-Out

IOH = 400 µA, IIH = 40 µA → 400/40 = 10
IOL = 16 mA, IIL = 1.6 mA → 16/1.6 = 10
Fan-Out = Min(10, 10) = 10 ✓

⚠️ Exceeding fan-out causes overloading → output voltage degrades → logic errors!

⑥ Power Dissipation

Power consumed per gate. Affects thermal design and battery life in portable systems.

PD(avg) = Icc(avg) × VCC
Icc(avg) = (ICCH + ICCL) / 2

VCC = supply voltage  |  ICCH = current when output HIGH  |  ICCL = current when output LOW

Ex: TTL Package with 4 NAND gates

VCC = 5V, ICCH = 1 mA, ICCL = 3 mA
Iavg = (1+3)/2 = 2 mA
PD/gate = 2 mA × 5V = 10 mW
Total (4 gates) = 4 × 10 = 40 mW ✓

⑦ Propagation Delay

Time taken for output to respond to a change in input. Limits maximum operating frequency.

tP(avg) = (tPHL + tPLH) / 2
ttotal = tpropagation × Nlevels

tPHL = delay when output goes High→Low
tPLH = delay when output goes Low→High
Nlevels = total number of logic levels in the path

Speed vs Accuracy

For high-speed circuits, maximum delay (worst case) is more critical than average. Design must guarantee timing even in worst case.

⑧ Noise Margin

Noise = undesirable signal superimposed on normal signal (DC or AC). Noise margin = max noise that can be added without causing a logic error.

NM = Min(VOH − VIH , VIL − VOL)
SymbolMeaning
VOHMin output voltage for logic HIGH
VIHMin input voltage recognized as HIGH
VOLMax output voltage for logic LOW
VILMax input voltage recognized as LOW

Design rule: VIL > VOL and VIH < VOH for valid noise margin.

① Bipolar Transistor

All bipolar IC families use NPN silicon transistors. Three operating regions:

RegionVBEVCEState
Cutoff< 0.6VOpenOFF (IB=IC=0)
Active0.6–0.7V>0.8VAmplifier
Saturation0.7–0.8V~0.2VON (switch)

hFE = DC current gain: IC = IB × hFE
In digital logic: transistor acts as ON/OFF switch (saturation/cutoff only).

Saturation condition: IB ≥ IC/hFE

② RTL — Resistor Transistor Logic

The earliest and simplest logic family.

  • Basic gate: NOR gate
  • VCC = 3.6 V
  • LOW level = 0.2 V
  • HIGH level = 1 – 3.6 V
  • Input resistors (~450 Ω) limit base current
  • Collector resistor (~640 Ω) pulls output HIGH
NOR operation

If any input = HIGH → that transistor saturates → output pulled LOW.
Only when ALL inputs = LOW → all transistors cut off → output = HIGH (via RC).

⚠️ Disadvantages: Poor noise margin, low fan-out, slow. Mostly obsolete.

③ DTL — Diode Transistor Logic

Improved on RTL by using diodes for input logic.

  • Basic gate: NAND gate
  • VCC = 5 V
  • LOW level = 0.2 V
  • HIGH level = 4 – 5 V
  • Input diodes (D1, D2…) replace resistors → better noise margin
  • Output transistor (Q1) switches the output
NAND operation (Y = (ABC)')

All inputs HIGH → diodes forward biased → enough base current → Q1 saturates → Y = LOW.
Any input LOW → that diode clamps base → Q1 cuts off → Y = HIGH.

Modified DTL: adds extra transistor Q2 to supply more base current to Q1 → increases fan-out significantly.
④ TTL — Transistor-Transistor Logic

The dominant bipolar family for decades. Evolved from DTL.

  • Basic gate: NAND gate
  • Multi-emitter input transistor (Q1) replaces DTL input diodes
  • Each emitter forms a pn junction (like DTL diodes)
  • LOW = 0.2 V  |  HIGH = 2.4 – 5 V
  • VCC = 5 V (standard)
TTL Sub-families

Standard → Low-power (L) → High-speed (H) → Schottky (S) → Low-power Schottky (LS) → Advanced Schottky (AS, ALS, F)

⑤ Totem-Pole TTL

Active pull-up replaces passive resistor RL → reduced propagation delay.

  • Q4 sits on top of Q3 → "totem-pole" naming
  • Output HIGH: Q4 ON, Q3 OFF → Q4 actively charges load
  • Output LOW: Q3 ON (sat), Q4 OFF → Q3 sinks current
  • Diode D1 between Q4 base and Q3 collector ensures Q4 is OFF when Q3 is saturated (provides voltage drop)
  • Cannot wire-AND totem-pole outputs directly!
Why faster?

Active Q4 "pushes" current into the load (fast charging). Passive RL can only let current "trickle" in → much slower.

⑥ Schottky & 3-State TTL
Schottky TTL
  • Schottky diode placed base-to-collector on each transistor
  • Diode = metal + semiconductor (Vf ≈ 0.25V) → prevents deep saturation
  • Less stored charge → faster switching
  • Reduces propagation delay without increasing power
Three-State (Tri-State) TTL
  • State 1: LOW (Q3 ON, Q4 OFF)
  • State 2: HIGH (Q4 ON, Q3 OFF)
  • State 3: High-Z (both OFF) — output disconnected
  • Enables bus sharing — multiple gates share one wire
  • Controlled by a separate Enable/Control input
⑦ ECL — Emitter-Coupled Logic
  • Non-saturated logic family → transistors never saturate → fastest switching
  • Propagation delay: 1–2 ns (fastest bipolar family)
  • Used in: mainframe computers, high-speed networking
  • Noise margin: only 0.3 V (very poor)
  • Very high power dissipation
  • High fan-out possible
  • Negative supply voltage (VEE = −5.2 V)

⚠️ Now largely obsolete — replaced by high-speed CMOS which achieves similar speed with far lower power.

⑧ MOS — Metal-Oxide Semiconductor

Unipolar (only one carrier type) transistor. Two types:

pMOS

p-channel transistor, n-type substrate. Gate(−) attracts holes to form channel. Slower.

nMOS

n-channel transistor, p-type substrate. Gate(+) attracts electrons. Faster than pMOS.

  • Low power consumption compared to bipolar
  • Small area on silicon → high density
  • Implements: Inverter, NAND, NOR gates
  • Slower than bipolar (TTL, ECL)
⑨ CMOS — Complementary MOS

Uses both pMOS and nMOS transistors in complementary pairs → most important family today.

  • Inverter: pMOS (pull-up) + nMOS (pull-down)
  • NAND: 2 pMOS in parallel + 2 nMOS in series
  • NOR: 2 pMOS in series + 2 nMOS in parallel
  • Near-zero static power — only draws current during switching
  • Excellent noise margin (~VDD/2)
  • VDD = 3.3V, 2.5V, 1.8V, 1.2V (modern)
Why CMOS dominates

Low power + high density + good noise immunity + scalability = used in all modern CPUs, GPUs, smartphones, FPGAs.
